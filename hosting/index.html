Smart Class Check-in & Learning Reflection

Midterm MVP project built with React Native (Expo), including complete pre-class check-in and post-class reflection with QR and geolocation evidence.

React Native + Expo QR + GPS Validation SQLite (native) localStorage (web) Firebase Hosting

Core Deliverables

Pre-Class Check-in

Collects student ID, name, mood, class QR code, and current location before class starts.

Post-Class Reflection

Requires the same QR code and records reflection summary plus confidence rating.

Integrity Rules

One check-in per day, QR matching between flows, and device-bound student ID lock.
